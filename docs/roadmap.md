# MVP Roadmap

## Week 1 - 기본 뼈대

- Supabase 프로젝트 생성
- DB 테이블 설계 및 RLS 정책 준비
- Next.js App Router 프로젝트 초기화
- 미용사 로그인 페이지 구현
- 손님 목록 / 검색 / 신규 등록 UI 구성
- 시술 종류 하드코딩 및 기본 버튼 배치

## Week 2 - 핵심 기능

- 시술 기록 추가 API 구현
- 시술 누적 횟수 계산
- 쿠폰 자동 발급 로직 구현
- 손님 상세 페이지에 시술 히스토리 표시
- 쿠폰 상태 관리 (active / used)

## Week 3 - 손님 공개 페이지 + 완성

- 손님 공개 링크 / QR 토큰 생성
- `c/[token]` 페이지 구현
- 손님용 스탬프 보드, 쿠폰 목록 UI 구현
- 미용사 화면 UX 다듬기
- Vercel 배포 및 QA

## 3일 컷이 가능한 조건

1. 범위를 MVP로 좁힐 것
2. 시술 종류는 초기에 최소 4~5개만 등록
3. UI는 shadcn/ui 컴포넌트 조합으로 빠르게 구성
4. Supabase RLS는 미용사용 admin 영역에만 적용
5. 손님 공개 페이지는 읽기 전용, 로그인 없이 접근
